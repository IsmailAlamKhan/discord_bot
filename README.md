## Installation
1. Clone the repository
2. Install the dependencies
```bash
dart pub get
```
3. Create a `.env` file in the root directory and add the following
```env
BOT_TOKEN=<YOUR_BOT_TOKEN>
PREFIX=<YOUR_BOT_PREFIX>
FOOTER_TEXT=<YOUR_FOOTER_TEXT>

```
